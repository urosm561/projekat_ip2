DrawRepeats

DrawRepeats is a tool for drawing repeats found with programs StatRepeats or Repeats. The program produces a latex document from the input file which is a resulting file from programs StatRepeats or Repeats with load output option.

Usage

DrawRepeats < input file name > < output file name > < left interval boundary > < right interval boundary > [optional arguments]
where
  • Input file name, minimal length, left interval boundary, right interval boundary are mandatory arguments.
  • Input file name is the name of the load file to be processed. This file is the result of StatRepeats or Repeats with output option load.
  • Left interval boundary specifies the starting position in the input sequence from where repeats results are observed.
  • Right interval boundary specifies the ending position in the input sequence to which the repeats results are observed.

Drawing repeats longer than given length

With option gel length only repeats greater or equal to given length are considered.

Drawing repeats only of given length

With option el length only repeats with equal to given boundary are considered. Options gel and el can not be combined.

Setup Instructions

Linux

To compile DrawRepeats execute the following command in the directory where you unpacked the archive:

     ./DrawRepeats.compile.sh

This command produces the DrawRepeats binaries.

Windows

DrawRepeats executable version (DrawRepeats.exe) is provided in this archive.
