MotifSearch

MotifSearch is a program for finding specified motifs in input sequence. The program also calculates the expected number of such motifs in random sequence of same length abd alphabet cardinality.

Usage

MotifSearch < input file name > < motif mask > [output file name]

where input file name and motif mask are mandatory arguments and output file name is an optional argument.

Input file name

Input file name is the name of the input file to be processed. If the file is not in the same directory as the program the whole path should be specified. This file is expected to be in fasta format. Fasta format means the first line starts with > and the sequence name is parsed from the first line. The actual text starts from the second line.

Motif Mask

Motif mask specifies the mask that all found motifs should satisfy as mentioned in definition 1.. Symbols used in specifying motif masks are:

  • any alphabet symbol a1,a2,.....,an. that specifies the letter that should be on that position.
  • . symbol that specifies any alphabet symbol can be on that position.
  • [a1,a2..an] that specifies that any of letters specified between brackets could be on that position.
  • [^a1,a2..an] that specifies that none of letters specified between brackets could be on that position.

Setup Instructions

Linux

To compile MotifSearch execute the following command in the directory where you unpacked the archive:

     ./MotifSearch.compile.sh

This command produces the MotifSearch binaries.

Windows

MotifSearch executable version (MotifSearch.exe) is provided in this archive. You will need to install the Visual C++ 2019 runtime in order to run it. It can be downloaded from Microsoft site.

To compile from source you will need the free Visual Studio 2019 Community Edition. Open MotifSearch.sln which is included in the archive and build the solution.
