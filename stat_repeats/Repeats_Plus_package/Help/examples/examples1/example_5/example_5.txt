Example 1. Calculate (dn) repeats with minimal length 10 and motif AC.T in Zaire ebolavirus</a> isolate Ebola virus/H.sapiens-tc/COD/1976/Yambuku-Mayinga nucleotide sequence (NCBI id NC_002549)
In place of . in the specified motif, both strings that represent the repeat will have the same letter. Statistical filtering is included by default. 

Input file : NC_002549.fasta
Syntax:      ./StatRepeats NC_002549.fasta 10 -motif AC.T
Output file: standard output (screen)




Example 2. Calculate (dn) repeats with minimal length 10 and motif A[^C]C[AT]T without probability filtering in Zaire ebolavirus</a> isolate Ebola virus/H.sapiens-tc/COD/1976/Yambuku-Mayinga nucleotide sequence (NCBI id NC_002549)

Input file : NC_002549.fasta
Syntax:      ./StatRepeats NC_002549.fasta 5 -motif A[^C]C[AT]T -noprob
Output file: standard output (screen)
