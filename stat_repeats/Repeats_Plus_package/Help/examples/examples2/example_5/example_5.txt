Example 1. Calculate (dn) repeats with minimal length 10 and motif AC.T in Zaire ebolavirus</a> isolate Ebola virus/H.sapiens-tc/COD/1976/Yambuku-Mayinga nucleotide sequence (NCBI id NC_002549)
In place of . in the specified motif, strings that represent the repeat can have different letters. 

Input file : NC_002549.fasta
Syntax:      ./Repeats NC_002549.fasta 10 -motif AC.T
Output file: standard output (screen)

Input file : NC_002549.fasta
Syntax:      ./Repeats NC_002549.fasta 5 -motif A[^C]C[AT]T 
Output file: standard output (screen)



Example 3. Calculate (dn) repeats with minimal length 15 and one mismatch in Zaire ebolavirus</a> isolate Ebola virus/H.sapiens-tc/COD/1976/Yambuku-Mayinga nucleotide sequence (NCBI id NC_002549)
In place of . in the specified motif, strings that represent the repeat can have different letters, so with option -motif . we are searching for repeats with one mismatch. 

Input file : NC_002549.fasta
Syntax:      ./Repeats NC_002549.fasta 10 -motif .
Output file: standard output (screen)
