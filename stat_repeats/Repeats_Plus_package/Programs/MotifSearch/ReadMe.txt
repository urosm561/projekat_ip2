Setup Instructions
In this archive all cpp and h files for MotifSearch are provided.

Linux

To compile MotifSearch execute the following command in the directory where you unpacked the archive:
	./MotifSearch.sh 

This command produces the MotifSearch binaries. In order for the above command to succeed you will need gcc 4.8.4 or later

Windows

MotifSearch executable version (MotifSearch.exe) is provided in this archive in folder Windows. You will need to install the Visual C++ 2019 runtime in order to run it. It can be downloaded from Microsoft cite (https://www.microsoft.com/en-us/download/details.aspx?id=48145).
To compile from source you will need the free Visual Studio 2019 Community Edition. Open MotifSearch.sln which is included in the archive Windows and build the solution.
