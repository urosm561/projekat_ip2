Use Instructions

In this archive StatRepeatsUi.py is provided that can be used with Python 3.

Windows executable (StatRepeatsUi.exe) is also provided for use under Windows. 

For working the program needs Repeats.exe and StatRepeats.exe to be in the same directory. The program will invoke one of the two programs depending on the input arguments.
RepeatsHelp.html should also be in the same directory.

Note: to use it with RepeatPlus graphical interface StatRepeats and Repeats programs must be locatexd in the same directory where RepeatsPlusUI is located, and named as "program_name[.exe]", without version and release qualificators.
